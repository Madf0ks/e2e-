Сценарий для тестирования сайта flowers:
1. Проверить функциональность сущности "Корзина":
    1.1. Добавление товаров
    1.2. Правильное считывание цены
    1.3. Удаление товаров
    1.4. Редактирование количества
2. Проверить наличие и функциональность каталога:
    2.1. Наличие каталога товаров
    2.2. Возможность добавить товар в корзину
    2.3. Перейти на другую страницу ( пагинация )
3. Проверить наличие и функциональность категорий
    3.1. В шапке должен быть подраздел с категориями товаров
    3.2. Ссылки должны быть не битыми и правильно перенаправлять на существующую страницу
4. Проверить функциональность поиска
    4.1. Наличие элемента поиска
    4.2. Любой запрос должен отдавать запрашиваемую информацию
5. Карточки товаров
    5.1. Содержание карточки: картинка, заголовок, цена, кнопка купить, баллы
    5.2. Правильная (  не битая ) ссылка, которая ведет на страницу товара
